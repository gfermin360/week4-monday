Explorando las Nuevas Etiquetas Semánticas en HTML5

Etiqueta <dialog>: creando modales simplificados

La etiqueta <dialog> permite crear modales o ventanas flotantes de forma nativa, sin la necesidad de usar JS y CSS. El atributo "open" indica si el modal es visible al cargar la página o no
